# Keylogger

This is a simple keylogger script written in Python. It captures keyboard inputs and logs them to a file.

## Disclaimer

**This tool is intended for educational purposes only.** Unauthorized use of keyloggers can have serious legal and ethical consequences. Always ensure you have explicit permission before monitoring any computer activity. Misuse of this tool is solely your responsibility.

## Installation

To run this script, you need to have Python installed. You also need the `pynput` library.

1.  **Install `pynput`:**
    ```bash
    pip install pynput
    ```

## Usage

1.  **Run the script:**
    Navigate to the directory containing `keylogger.py` and run:
    ```bash
    python keylogger.py
    ```
    (Note: If you encountered issues running `keylogger.py` directly, please refer to the previous conversation for troubleshooting steps, such as checking antivirus, file permissions, or trying a different filename.)

2.  **Stop the keylogger:**
    Press the `Esc` key on your keyboard.

3.  **View Logs:**
    Key presses will be logged to a file named `keylog.txt` in the same directory where the script is run.
